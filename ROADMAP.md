# Roadmap

## v0.0 — Continuity Exists

Event → Timeline → Lineage

Goal:
Prove continuity can be visualized.

Deliverables:

* Events table
* Create Event screen
* Timeline screen
* Lineage screen

## v0.1 — Receipts

Goal:
Prove continuity can be validated.

Deliverables:

* Receipts table
* Issue Receipt action
* Receipt Viewer

## v0.2 — File Continuity

Goal:
Prove continuity can represent real artifacts.

Deliverables:

* File Tree
* Monaco Editor
* File.Opened events
* File.Saved events

## Definition of Done

A user can create an event, inspect lineage, issue a receipt, edit a file, and observe the resulting continuity chain.
