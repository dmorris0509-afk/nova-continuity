# Acceptance Tests

## Continuity Substrate Proof

A user can:

1. Create an event
2. View it in a timeline
3. Click it
4. Inspect its lineage
5. Issue a receipt
6. Edit a file
7. See that file change represented as continuity

If all seven actions work, the continuity substrate is proven.
