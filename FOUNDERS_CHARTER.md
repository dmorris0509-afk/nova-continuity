# Purpose

Nova exists to build continuity-native systems that preserve context, traceability, governance, and institutional memory across generations.

The founders intend to create durable technology, intellectual property, and organizational infrastructure that can continue creating value long after the original founders are gone.

---

# Founder Relationship

The founders recognize each other as equal partners in the creation and stewardship of Nova.

The project is built through the combination of:

* Vision
* Architecture
* Engineering
* Governance
* Execution
* Long-term stewardship

Neither founder is considered subordinate to the other.

---

# Shared Vision

We believe:

* Continuity matters.
* Governance matters.
* Memory matters.
* Systems should be explainable.
* Systems should be auditable.
* Systems should outlive individual contributors.

Nova is intended to become a continuity-native platform capable of supporting future generations of builders, operators, organizations, and communities.

---

# Core Principles

## 1. Continuity First

Every meaningful state change should be traceable.

## 2. Governance by Design

Governance should be built into systems rather than added later.

## 3. Preservation of Knowledge

Institutional memory is a strategic asset.

## 4. Long-Term Thinking

Short-term decisions should support long-term sustainability.

## 5. Shared Stewardship

The founders are custodians of the mission, architecture, and future direction of the project.
